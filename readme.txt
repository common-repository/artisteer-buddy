== Description ==

A collection of functions to better Artisteer built themes.

= Features: = 
* Removes HTML attributes displayed under the comment field. 
* Removes "Comments are closed" text from under comment field. 
* Disables comments on pages. 
* Adds Thumbnails to RSS feeds. 
* Adds simple pagination to theme. (No code edit required) (CSS filed included).

== Installation ==

To install the plugin:

1. Upload the zipfile to the `/wp-content/plugins/` directory
2. Extract and remove it
3. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= What if I don't want / need one of these options? =

The plug-in file is organized and features are displayed under the standard php double backslash comments. Just delete the code for the feature in between the mark-up.
